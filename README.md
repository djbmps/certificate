# ระบบออกเกียรติบัตร

ออกเกียรติบัตรทีละร้อยใบใน 5 นาที — ใส่รายชื่อครั้งเดียว ระบบสร้างให้ครบทุกคนอัตโนมัติ

**ใช้งานได้ที่:** https://djbmps.github.io/certificate/

---

## คุณสมบัติ

- **ใส่รายชื่อครั้งเดียว ออกครบทุกคน** — นำเข้าจาก Excel/CSV วางรายชื่อทีเดียวหลายคน หรือพิมพ์ในตารางเอง เพิ่มคอลัมน์ได้ตามต้องการ
- **พื้นหลังสำเร็จรูป 8 แบบ** — เลือกใช้ได้ทันที หรืออัปโหลดแบบของหน่วยงานเอง หรือออกแบบใน Canva
- **ตราสัญลักษณ์และลายเซ็น** — ใส่โลโก้ได้หลายอัน ลากจัดตำแหน่งอิสระ · อัปโหลดลายเซ็นสแกนของผู้ลงนามแต่ละคน
- **ฟอนต์ไทย 16 แบบ** — ปรับขนาด สี ตัวหนา และตำแหน่งได้ทุกข้อความ
- **เลขที่เกียรติบัตร** — รันเลขอัตโนมัติ หรือกำหนดเองจากไฟล์รายชื่อ
- **เลขไทย/อารบิก** — สลับรูปแบบตัวเลขในเกียรติบัตรได้
- **ส่งออก PNG / JPEG / PDF** — PDF รวมไฟล์เดียวหรือแยกรายคน พร้อมพิมพ์
- **บันทึกรูปแบบไว้ใช้ซ้ำ** — เก็บเป็นไฟล์ `.certproj` ครั้งหน้าเปิดใช้ได้ทันที

## ความเป็นส่วนตัว

ระบบประมวลผลในเครื่องผู้ใช้ทั้งหมด **ไม่มีการส่งรายชื่อหรือข้อมูลใด ๆ ออกนอกเครื่อง** ไม่มีเซิร์ฟเวอร์ ไม่มีฐานข้อมูล จึงปลอดภัยตาม พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล (PDPA)

## การใช้งาน

**แบบออนไลน์** — เปิดลิงก์ด้านบนได้เลย ใช้ได้ทั้งมือถือและคอมพิวเตอร์ ไม่ต้องสมัคร ไม่ต้องติดตั้ง

**แบบออฟไลน์** — ดาวน์โหลดไฟล์ `ระบบออกเกียรติบัตร.html` แล้วเปิดด้วยเบราว์เซอร์ ทำงานได้โดยไม่ต้องต่ออินเทอร์เน็ต (ฟอนต์จะเปลี่ยนไปใช้ของเครื่องแทน)
